Formatted reports

Formatted reports are summaries of selected project information in a format ready to print or export as a Word file.

This folder lists configurations for formatted reports ( does not save the resulting reports themselves). There are a number of default report configurations and you can create new ones as well.

Double-click a report configuration in the list to run it, showing the resulting report in the (note that some reports are configured to add filters when run).

Because saves report configurations, the same formatted report can produce reports with different content if run at different times.
