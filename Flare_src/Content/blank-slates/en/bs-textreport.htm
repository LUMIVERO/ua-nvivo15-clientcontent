Text reports

Text reports are summaries of selected project information that you can export as text, Excel or XML files for analysis in other software packages.

This folder lists configurations for text reports ( does not save the resulting reports themselves). There are a number of default report configurations and you can create new ones as well. The configuration options are the same as for formatted reports—only the output format is different.

Double-click a report configuration in the list to create a report (note that some reports are configured for you to add filters when run).

Because saves report configurations, the same text report can produce reports with different content if run at different times.
