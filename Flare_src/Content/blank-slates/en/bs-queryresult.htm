Query results are saved results from text searches and coding, matrix coding, crosstab, and compound queries (you cannot save query results from word frequency, coding comparison or group queries).

Query results are saved results from text searches and coding, matrix coding, and crosstab queries (you cannot save query results from word frequency or coding comparison queries).

Query results are the output of a query that was run at a specific time. If query results are opened at a later time, when the project has changed, the results are automatically checked and entries that are no longer valid removed. No new results are added, however.
