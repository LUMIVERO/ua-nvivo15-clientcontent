Coding matrices

Coding matrices are results tables from matrix coding queries.

The rows and columns of coding matrix tables are selected project items (e.g. files, cases, codes, classifications) or attributes. The cells show information about intersecting coding between the relevant items or attributes, e.g. the number of coding references, words coded, or cases coded.

You can color-code table cells to quickly identify where most coding occurred. Double-click cells to view the coded data.

Coding matrices show only the results tables created by matrix coding queries. You can also save results from these queries in the Query Results folder, where they include the query criteria as well as the results tables.
