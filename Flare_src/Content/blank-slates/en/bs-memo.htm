Memos

Memos are documents you create to keep notes about any aspects of your project.

Memos are kept separate from source data files but can be linked to these files or other project items that they relate to (in one-to-one relationships). Memos can contain text, images and tables and be coded and given attributes (under file classifications) like data files.

Optionally create subfolders to organize your memos.
