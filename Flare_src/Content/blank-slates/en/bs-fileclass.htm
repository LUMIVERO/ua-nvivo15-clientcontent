File classifications

File classifications are collections of attributes (descriptive information) about your data files.

For example, for photos you might want to record where and when they were taken, and by who. This collection of attributes: location, date, and author, will constitute one file classification which you could name 'Photos'.

Other types of data file—for example, interview transcripts or newspaper articles—can all have their own file classifications with attributes relevant to those types of file.
