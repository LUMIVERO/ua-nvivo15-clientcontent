Relatórios formatados

Os relatórios formatados são resumos das informações selecionadas do projeto em um formato pronto para imprimir ou exportar como um arquivo do Word.

Esta pasta lista as configurações para os relatórios formatados (o não salva os relatórios resultantes). Existem várias configurações de relatório padrão e você também pode criar novas.

Clique duas vezes em uma configuração de relatório na lista para executá-la, mostrando o relatório resultante no (observe que alguns relatórios estão configurados para adicionar filtros quando são executados).

Uma vez que o salva as configurações do relatório, o mesmo relatório formatado poderá produzir relatórios com conteúdo diferente se for executado em momentos diferentes.
