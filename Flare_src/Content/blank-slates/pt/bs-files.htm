Arquivos

Todos os arquivos de dados importados para o projeto são armazenados na pasta Arquivos. Você também pode criar novos documentos aqui.

Opcionalmente, crie subpastas para organizar seus dados. Por exemplo, reúna todas as entrevistas em uma pasta.

  • Para importar arquivos localizados no seu computador, acesse a guia Importar e use os comandos Arquivar, Pesquisar ou NCapture. Você também pode arrastar e soltar arquivos nesse espaço.
  • Para arquivos que você enviou para o com , use o comando Coletar arquivos do Office na guia Módulos.
