Relatórios de texto

Os relatórios de texto são resumos das informações selecionadas do projeto que você pode exportar como arquivos de texto, Excel ou XML para análise em outros pacotes de software.

Esta pasta lista configurações para relatórios de texto (o não salva os relatórios resultantes). Existem várias configurações de relatório padrão e você também pode criar novas. As opções de configuração são as mesmas dos relatórios formatados, apenas o formato de saída é diferente.

Clique duas vezes em uma configuração de relatório na lista para criar um relatório (observe que alguns relatórios estão configurados para você adicionar filtros quando executados).

Uma vez que o salva as configurações do relatório, o mesmo relatório de texto poderá gerar relatórios com conteúdo diferente se for executado em momentos diferentes.
