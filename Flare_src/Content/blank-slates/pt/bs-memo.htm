Memos

Os memos são documentos que você cria para manter anotações sobre todos os aspectos do seu projeto.

Os memos são mantidos separados dos arquivos de dados de origem, mas podem ser vinculados a esses arquivos ou a outros itens de projeto relacionados (em relacionamentos de um para um). Os memos podem conter texto, imagens e tabelas e ser codificados e receber atributos (nas classificações de arquivos) como arquivos de dados.

Opcionalmente, crie subpastas para organizar seus memos.
