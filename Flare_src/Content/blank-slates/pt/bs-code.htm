Códigos

Normalmente, os códigos são temas que você identifica em seus dados.

Os códigos coletam, em um só lugar, todo o texto, imagem (ou partes de imagens) e intervalo de tempo de áudio e vídeo que você define como relevantes para os temas identificados.

Todos os códigos são salvos aqui. Você pode criar subpastas para ajudar a organizá-los. Você também pode criar hierarquias de código. Por exemplo, um código "Ensino" pode ter códigos descendentes "Primário", "Secundário" e "Superior".
