Rapports formatés

Les rapports formatés sont des récapitulatifs des informations de projet sélectionnées dans un format prêt à imprimer ou à exporter sous forme de fichier Word.

Ce dossier répertorie les configurations des rapports formatés ( n'enregistre pas les rapports résultants eux-mêmes). Un certain nombre de configurations de rapport par défaut sont disponibles ; vous pouvez également en créer de nouvelles.

Double-cliquez sur une configuration de rapport dans la liste pour l'exécuter et afficher le rapport résultant dans la (notez que certains rapports sont configurés pour ajouter des filtres lors de leur exécution).

Du fait que enregistre les configurations de rapports, le contenu d'un rapport formaté peut changer selon le moment auquel il est exécuté.
