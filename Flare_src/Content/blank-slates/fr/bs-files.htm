Fichiers

Tous les fichiers de données que vous importez dans le projet sont stockés dans le dossier Fichiers. Vous pouvez également y créer de nouveaux documents.

Créez éventuellement des sous-dossiers pour organiser vos données, par exemple, regroupez tous les entretiens dans un dossier.

  • Pour importer des fichiers situés sur votre ordinateur, allez dans l'onglet Importer et utilisez les commandes Fichiers, Enquête ou NCapture. Vous pouvez également faire glisser et déposer des fichiers dans cet espace.
  • Pour les fichiers que vous avez envoyés à avec , utilisez la commande Collecter des fichiers Office dans l'onglet Modules.
