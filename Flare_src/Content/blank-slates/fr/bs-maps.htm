Cartes

Les cartes mentales , les cartes de projet et les cartes conceptuelles sont de différents types de diagrammes que vous pouvez créer pour montrer les relations entre les idées et/ou les éléments du projet. Vous pouvez enregistrer toutes les cartes que vous créez dans ce dossier.

Créez éventuellement des sous-dossiers pour organiser vos cartes.
