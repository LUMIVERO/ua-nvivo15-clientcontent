Cas

Les cas sont les personnes, les lieux, les organisations, les événements ou d'autres entités qui constituent les unités d'analyse d'un projet.

Selon la nature de votre recherche, vous n'aurez peut-être pas besoin de cas ; cependant, l'analyse des données en termes d'attributs tels que le sexe ou le groupe d'âge nécessite des cas, ainsi que des classifications de cas.

Tous les cas sont enregistrés ici : vous pouvez créer des sous-dossiers pour les organiser. Vous pouvez également créer des hiérarchies de cas. Par exemple, pour une étude sur les universités, vous pouvez créer les facultés en tant que cas « enfants » dans chacune d'elles.
