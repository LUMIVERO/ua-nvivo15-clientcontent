Rapports texte

Les rapports texte sont des récapitulatifs des informations de projet sélectionnées que vous pouvez exporter sous forme de fichiers texte, Excel ou XML à des fins d'analyse dans d'autres logiciels.

Ce dossier répertorie les configurations des rapports texte ( n'enregistre pas les rapports résultants eux-mêmes). Un certain nombre de configurations de rapport par défaut sont disponibles ; vous pouvez également en créer de nouvelles. Les options de configuration sont les mêmes que celles des rapports formatés. Seul le format de sortie est différent.

Double-cliquez sur une configuration de rapport dans la liste pour créer un rapport (notez que certains rapports sont configurés pour que vous leur ajoutiez des filtres lors de leur exécution).

Du fait que enregistre les configurations de rapports, le contenu d'un rapport texte peut changer selon le moment auquel il est exécuté.
