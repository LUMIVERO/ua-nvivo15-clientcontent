Codes

En règle générale, les codes sont des thèmes que vous identifiez dans vos données.

Les codes collectent et centralisent tout le texte, toutes les images (ou parties d'images) et toutes les plages audio et vidéo que vous trouvez pertinentes pour les thèmes que vous identifiez.

Tous les codes sont enregistrés ici : vous pouvez créer des sous-dossiers pour les organiser. Vous pouvez également créer des hiérarchies de codes : par exemple, un code « Éducation » peut avoir des codes enfants « Primaire », « Secondaire » et « Supérieure ».
