Dateien

Alle Dateien, die Sie in das Projekt importieren, werden im Dateiordner gespeichert. Hier können Sie auch neue Dokumente erstellen.

Sie können wahlweise Unterordner anlegen, um Ihre Daten zu organisieren. So können Sie beispielsweise alle Interviews in einem Ordner zusammenfügen.

  • Um Dateien auf Ihrem Computer zu importieren, gehen Sie zur Registerkarte Importieren und verwenden Sie die Befehle Dateien, Umfrage oder NCapture. Sie können die Dateien auch per Drag & Drop in diesen Bereich ziehen.
  • Bei Dateien, die Sie an mit gesendet haben, verwenden Sie den Befehl Office-Dateien erfassen auf der Registerkarte Module.
