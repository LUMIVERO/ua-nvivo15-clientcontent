Anotaciones

Las anotaciones son notas para el contenido seleccionado en archivos de datos o memos.

Este panel muestra todas las anotaciones en el proyecto: haga doble clic en una anotación para abrir el archivo relevante y ver la anotación en la parte inferior de (si no puede ver las anotaciones, marque Anotaciones en la cinta ) en una pantalla emergente junto al contenido seleccionado (si no puede ver las anotaciones, haga clic en Anotaciones en la parte superior de )

Puede crear anotaciones para selecciones de texto, para regiones dentro de imágenes y PDF, y para períodos de tiempo en archivos de video y audio.
