Elementos externos

Los elementos externos son documentos proxy creados en para representar fuentes de datos que no se pueden importar.

Por ejemplo, puede crear elementos externos para libros o películas a los que no puede acceder en forma digital. Use los elementos externos para resumir los materiales de fuentes; también puede transcribir o copiar secciones. El contenido de los elementos externos se puede codificar de la misma manera que el contenido de los archivos de datos importados.
